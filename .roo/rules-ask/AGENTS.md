# Project Documentation Rules (Non-Obvious Only)

- Frontend code is in clientapp/ directory with Next.js App Router structure
- Backend API is in ServerApp/ directory using .NET 8
- Docker configuration is in docker-compose/ directory
- Images are stored in /public/ directory
- Painting categories are defined in /src/app/models/paintingCategories.ts
- ArtCarousel component is in /src/components/ directory
- Next.js Image component is used for optimized image loading
